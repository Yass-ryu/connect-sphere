# ConnectSphere – Full Stack Social Networking Platform

## 🚀 Overview
ConnectSphere is a full-stack social networking application built using the MERN stack. It allows users to create posts, interact with content, and manage social connections in a dynamic and responsive environment.

The project demonstrates full-stack development skills including frontend UI design, backend API development, authentication, and database management.

## ✨ Features

- User authentication and authorization (JWT-based)
- Create, update, and delete posts
- Like and interact with posts
- Dynamic and responsive feed
- RESTful API architecture
- Secure backend implementation
